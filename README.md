# Wireless Tech Solution USA — Frontend (React + Vite)

This is a minimal React + Vite scaffold for the Wireless Tech Solution USA LLC website. It includes homepage, about, services and contact pages with components and styles.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

Notes
- Replace the Google Maps API key in `src/pages/Contact.jsx` (the `iframe` src) with your own key, or switch to a static map.
- Add real phone/email and hero background at `src/assets/tech-bg.jpg` (placeholder path used in CSS).
